
# 🎮 Unity UI Navigation System

## 🌟 About the Project
This is my first User Interface (UI) navigation system created in **Unity**. As an aspiring **Technical Artist** with a background in Graphic Design, my goal is to create smooth, logical, and visually appealing menus for games. 

This project demonstrates a clean and functional menu flow using **C#** and Unity's UI Canvas system.

## ✨ Features
* **Main Menu:** Interactive interface with "New Game" and "Settings" buttons.
* **Settings Panel:** Includes a functional volume/setting slider and a "Back" button to return to the main screen.
* **Game Screen:** A "Play" screen with an option to navigate back to the main menu.
* **Clean Logic:** Optimized navigation without breaking the game flow.

## 🛠️ Tools Used
* **Game Engine:** Unity
* **Language:** C#
* **Design Focus:** User Experience (UX) & UI Flow
